# Library Management System

A professional library management system built with Node.js, Sequelize ORM, and MySQL.  
Provides a complete solution for managing authors, books, members, and borrowing records with proper database relationships and data validation.

---

## Features

- Full CRUD operations for all entities  
- One-to-Many and Many-to-Many relationships  
- Data validation (email format, ISBN length, date ranges, price limits)  
- SSL/TLS support for secure MySQL connections  
- Automated database migrations with test data  
- Custom business logic methods  
- Structured and professional error handling  

---

## Tech Stack

- Backend: Node.js 18+  
- ORM: Sequelize 6  
- Database: MySQL 5.7+ (mysql2 driver)  
- Configuration: dotenv  
- Security: SSL/TLS support
